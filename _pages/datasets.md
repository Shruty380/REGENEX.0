Regenerative Transcriptomics Dataset Resources

This section documents publicly available transcriptomic datasets utilised for computational regeneration modelling.

Key Dataset Categories

🧬 Limb regeneration RNA-seq datasets (axolotl, zebrafish)

🧬 Neural injury and repair transcriptomic datasets

🧬 Cardiac regeneration expression profiles

🧬 Fibrosis and degenerative disease control datasets

Primary Data Source Platforms

• NCBI Gene Expression Omnibus (GEO)
• EMBL-EBI ArrayExpress
• LINCS perturbation transcriptomics database

Computational Usage

These datasets are integrated into the REGENEX pipeline for:

• Differential expression modelling
• Pathway reconstruction
• Drug signature matching
• Network-level regenerative potential scoring

This curated dataset framework ensures reproducible computational biology research workflows.
